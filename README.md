# AI Financial Intelligence Dashboard

An AI-powered financial analytics platform built using **n8n**, **OpenAI**, **Google Sheets**, and **Looker Studio**.

The system automatically processes bank statements, categorizes expenses using AI, and generates intelligent financial insights through automated workflows and dashboards.

---

## Features

* Excel bank statement ingestion
* AI-powered expense categorization
* Automated workflow orchestration using n8n
* Monthly AI-generated financial insights
* Savings health score
* Interactive Looker Studio dashboard
* Google Sheets integration

---

## Architecture

Bank Statement Upload
↓
n8n Workflow Automation
↓
OpenAI Expense Categorization
↓
Google Sheets Storage
↓
AI Financial Insights Generation
↓
Looker Studio Dashboard

## Workflows

### 1. Expense Categorization Workflow

* Reads Excel bank statements
* Extracts transaction data
* Categorizes transactions using AI
* Stores structured data in Google Sheets

### 2. Monthly Financial Insights Workflow

* Aggregates monthly spending
* Generates AI-powered financial recommendations
* Calculates savings health score
* Updates dashboard insights automatically
